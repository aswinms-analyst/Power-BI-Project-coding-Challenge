# Global Bike-Sharing Station Analytics 🚴‍♂️📊
# Overview
This project presents an interactive data analytics dashboard that explores global bike-sharing station performance across multiple cities. The dashboard provides insights into availability, utilization, and geographic distribution of bike stations, helping identify usage trends and operational efficiency.

# Key Features
Station Status: Pie chart showing open vs. closed stations.

KPIs: Total bike stands (54K) and total available bikes (18K).

City-Level Analysis: Bar chart of available bikes by contract name (e.g., Dublin, Brussels, Toyama).

Station-Level Details: Table view with bike stands, available bikes, and utilization % for Toyama, Japan.

Trend Analysis: Line chart of monthly availability and utilization rates.

Geospatial Insights: Map visualization of station locations across Europe and Asia.

# Data Highlights
Utilization % helps measure demand vs. supply at each station.

Seasonal trends reveal fluctuations in bike availability across months.

Geographic mapping enables comparison of station density and accessibility.

# Tech Stack
Power BI / Tableau (for dashboard design)

Data Cleaning & Transformation (CSV, Pandas)

Visualization: Pie, bar, line, map, and tabular reports

# Business Value
Supports urban mobility planning by identifying underutilized stations.

Helps optimize bike distribution and maintenance schedules.

Provides data-driven insights for city authorities and bike-sharing operators.
